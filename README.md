# Medical image analysis for pneumonia detection

A simple AI system that detects pneumonia from chest X-ray images.

## Quick Start
1. `pip install -r requirements.txt`
2. `python training.py` (if no model exists)
3. `streamlit run app_enhanced.py`

## Files
- `app_enhanced.py` - Web interface
- `training.py` - Train AI model  
- `check_accuracy.py` - Test accuracy
- `pneumonia_model.h5` - Trained AI
